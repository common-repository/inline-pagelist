Creates a list of pages or posts within a post. Select a parent page, category or tag and get a related content list inside your post.

== Description ==

With this plugin, you can put list of pages inside your post. Perhaps you want a list of subpages or the last post for a category or tag as a related content. This plugin does that. 

It's so easy to use, once activated, just insert an special tag within you post to select the parent page, the category or tag and some optional options. Also some functions for theme developers are provided.

= You can get three types of list: =
 
* A list of child pages from a given parent page. 
* The latest posts for a given category. 
* The latest post with a tag. 

= The plugin can be used in three different ways: =

1. By using a <a href="http://wiki.alkivia.org/pagelist/tags">special short code</a> in your post or page.
1. By using <a href="http://wiki.alkivia.org/pagelist/fields">cutom fields</a> in the post or page. (Requires to include a function to your template).
1. By <a href="http://wiki.alkivia.org/pagelist/functions">calling some functions</a> in your templates.

= Features: =

* Insert a list of posts or pages within your post or page.
* Select the depth level for the page list.
* Insert a list of posts with a particular tag or from a given category inside any post or page.
* Available functions for theme developers.
* Customize by using CSS styles.

= Languages included: =

== Installation ==

Before updating the plugin, save your style.css file in case you have modified it!

= Installing the Plugin =

1. Unzip the plugin archive
1. Upload the plugin folder to the WordPress plugins directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. There are no configuration options for the plugin
1. Enjoy it!.
